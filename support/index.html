OrderPilot Pro - WooCommerce Order Notification Manager

Getting Started

To start receiving and managing your store's orders, connect the App to your WooCommerce store using the OrderPilot Pro WordPress plugin:

  1. Install and activate the OrderPilot Pro plugin on your WordPress site.
  2. Open WordPress Admin > OrderPilot Pro > Notifications.
  3. Copy the 6-digit Store Code shown on that page.
  4. Open the OrderPilot Pro app, go to Settings, enter your name and the Store Code, then tap Connect to Store.
  5. All staff using the same Store Code will receive order notifications on their devices.

Frequently Asked Questions

I'm not receiving order notifications. What should I check?

  • Make sure notifications are enabled for OrderPilot Pro in your phone's system settings.
  • Disable battery-optimization / background-restriction for the App so it can stay connected (varies by phone manufacturer).
  • Confirm the Store Code in the App matches the one shown in your WordPress admin.
  • Check that your phone has an active internet connection.

I get a "Proxy or VPN detected" message. What does this mean?

For security, OrderPilot Pro blocks store data requests while a VPN or proxy is active on your device. Disable any VPN or proxy app and reopen OrderPilot Pro.

"Store code was not found" - what should I do?

Double-check the 6-digit code from WordPress Admin > OrderPilot Pro > Notifications and make sure the OrderPilot Pro plugin is active on your site. Codes are case-sensitive and must be entered exactly as shown.

How do I disconnect my store or switch accounts?

Go to Settings and use the disconnect option to remove the saved connection from your device. You can then connect again with a different Store Code.

The app crashed - how can I report it?

Reopen the App. If a crash was recorded, you'll see a dialog with the error details, or you can find it under Developer > Crash Log. Tap "Copy" and send it to us using the contact details below so we can fix it quickly.

Is my store data shared with anyone else?

No. Your data is only used to operate the App's core features. See our Privacy Policy for full details.

Still Need Help?

If your issue isn't covered above, reach out to us directly. Please include your store domain, a description of the issue, and (if relevant) a copy of the crash log.
